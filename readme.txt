Projenin tüm gereksinimlerini hazırlamış olup veritabanının da full backup dosyasını ekte iletmekteyim.

Projeyi çalıştırdıktan sonra admin arayüzüne erişmek isterseniz giriş yapma alanından:
admin@telefondefteri.com maili ve Admin=1 şifresi ile giriş yapabilirsiniz.

Proje asp.net core 2.2 ile geliştirilmiştir. 
Veritabanlarına erişim için entity framework core (codefirst) kullanılmıştır.
Veritabanı olarak SQL Server kullanılmıştır.

Ayrıca proje github linki şu şekildedir: https://github.com/denizdikbiyik/TelefonRehberiProject
